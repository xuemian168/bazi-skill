# Xiangfa Safety Rewrites

Source basis: `SRC-PROJ-SAFETY`, `analysis-methods.md`, and report safety policy. Use this for all user-facing 象法 / 盲派象法 text.

## Rewrite Table

| Unsafe wording | Use instead |
|---|---|
| 克父母 / 克妻 / 克夫 / 克子 | 家庭或亲密关系中更容易出现边界、照顾责任、沟通节奏或现实压力议题。 |
| 必离婚 / 必二婚 | 关系稳定性需要经营；若运年触发冲刑，适合提前处理沟通、财务和生活安排。 |
| 血光之灾 / 大灾 | 该阶段波动和冲突信号较强，宜降低冒险、注意安全和流程检查。 |
| 牢狱 / 官非必来 | 规则、合同、审批或责任压力较突出，宜重视合规、文书和沟通记录。 |
| 重病 / 绝症 | 身心压力或作息失衡风险需要关注；具体健康问题应咨询专业医生。 |
| 必发财 / 暴富 | 资源经营或变现机会增强，但结果取决于能力、市场、执行和风险控制。 |
| 破产 | 财务波动或资源分配压力上升，宜控制杠杆、预算和合作边界。 |
| 子女不孝 / 无子 | 后辈、子女或长期项目议题需要更多耐心和清晰边界；不做生育结论。 |
| 犯小人 | 合作中的信息不对称、角色冲突或隐藏阻力需要被显性化处理。 |

## Required Caveats

- For health: use wellness and stress language only; never diagnose.
- For finance: discuss risk style and resource management; never recommend securities or guarantee return.
- For relationships: describe dynamics and repair paths; never call a relationship doomed.
- For timing: rank windows and explain tradeoffs; never guarantee outcome.

## Tone Rules

- Prefer concrete but bounded language.
- Name the evidence before the image.
- Add a repair path for every conflict image.
- If the only available wording would be frightening or deterministic, suppress it and return `evidence_gap`.
