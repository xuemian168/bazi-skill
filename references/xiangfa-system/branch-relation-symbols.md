# Branch-Relation Symbols

Source basis: `SRC-PROJ-BZ` branch relation lists, `SRC-PROJ-METHODS` branch-relation weather method, and `SRC-PROJ-COMPAT` branch interaction matrix. Use only when the relation is computed or supplied.

## Core Relations

### `XF-BR-HE-001`

- Feature: 六合, supported 三合, or 三会.
- Symbol scope: binding, cooperation, affinity, shared direction, easier repair.
- Risk nuance: binding can also mean entanglement, obligation, or slow separation.
- Safe wording: "合象偏向连接与牵引，不等于必然顺利".

### `XF-BR-CHONG-001`

- Feature: 冲.
- Symbol scope: movement, relocation, change, pressure to decide, conflict, visibility through disruption.
- Risk nuance: the same冲 can create breakthrough or instability depending on Da Yun, event type, and practical constraints.
- Safe wording: "冲主动象/变象，不能单独断灾".

### `XF-BR-XING-001`

- Feature: 刑.
- Symbol scope: repeated friction, sensitivity, self-pressure, rules rubbing against habits, unresolved pattern.
- Risk nuance: easy to become chronic annoyance rather than one obvious event.
- Safe wording: "刑更像反复卡点，需要制度化修复".

### `XF-BR-HAI-001`

- Feature: 害.
- Symbol scope: hidden resistance, mismatched assumptions, indirect obstruction, private resentment.
- Risk nuance: often less visible than 冲; needs communication and explicit expectations.
- Safe wording: "害象不宜讲成暗害他人，只能说隐性阻力".

### `XF-BR-PO-001`

- Feature: 破.
- Symbol scope: loosening, breakage, dismantling, old arrangement becoming hard to maintain.
- Preconditions: only use if the host project computes or supplies 破. Current base reference does not list a complete破 matrix.
- Safe wording: "破象偏拆解旧结构；若未计算该关系，输出 evidence_gap".

### `XF-BR-TAOHUA-001`

- Feature: 子午卯酉 or project-supplied桃花/visibility trigger.
- Symbol scope: visibility, attraction, social attention, romance trigger, audience response.
- Risk nuance: visibility is not stability; attraction is not commitment.
- Safe wording: "桃花更像曝光和互动增强，不等于关系结果".

## Weighting

- Day branch relation has high relationship weight.
- Month branch relation has high environment/career-family rhythm weight.
- Year branch relation usually describes background/social layer.
- Hour branch relation needs birth-time confidence.
- Stacked relations raise attention but still need Da Yun / Liu Nian / event context.
